# Homebrew Tap for xToolbox

This is the official Homebrew tap for xToolbox.

## Installation

```bash
# Add the tap
brew tap rainx/tap

# Install xToolbox
brew install --cask xtoolbox
```

## One-line install

```bash
brew install --cask rainx/tap/xtoolbox
```

## Update

```bash
brew upgrade --cask xtoolbox
```

## Uninstall

```bash
# Uninstall the app
brew uninstall --cask xtoolbox

# Remove all data (optional)
brew uninstall --cask --zap xtoolbox
```

## What is xToolbox?

xToolbox is a personal Mac toolbox application that replicates useful features from paid apps:

- **Clipboard Manager** - Track clipboard history with text, images, files (Alt+Cmd+V)
- **Screenshot Snap** - Beautify screenshots with shadows and rounded corners (Alt+Cmd+A)
- More tools coming soon...
